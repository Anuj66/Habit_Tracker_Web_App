## How to Run
1.  **Backend**:
    ```bash
    cd server
    node index.js
    ```
    The server will run on `http://localhost:5000`.

2.  **Frontend**:
    ```bash
    cd client
    npm run dev
    ```
    The client will run on `http://localhost:5173` (or similar).
